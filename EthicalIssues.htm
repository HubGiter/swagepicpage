Ethical Issues

  1. Is it ok to read the documents and look at the graphics files that are stored on users' computers or in their directories on the file server?
  2. Is it ok to use screen capture programs so you can see everything that's displayed?
  3. is it ok to place key loggers on machines on the network to capture everything the user types?
  4. Is it ok to read employees' e-mail as a security measure to ensure that sensitive company information isn't being disclosed?
  5. Is it ok to read employees' e-mail to ensure that company rules (for instance, against personal use of the e-mail system) aren't being violated?
